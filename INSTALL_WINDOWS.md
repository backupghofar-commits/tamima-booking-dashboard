# 🪟 Panduan Install TAMIMA Booking Dashboard di Windows

## 📥 Cara Install (Mudah - 3 Langkah)

### Langkah 1: Buka URL Aplikasi

Buka aplikasi di **Microsoft Edge** atau **Google Chrome**:
```
https://USERNAME.github.io/tamima-booking-dashboard/
```

### Langkah 2: Install sebagai App

#### Cara A: Klik Tombol "Install App" di header
1. Cari tombol ungu **"⊕ Install App"** di pojok kanan atas aplikasi
2. Klik tombol tersebut
3. Klik **"Install"** di popup yang muncul

#### Cara B: Via Address Bar
1. Lihat ujung address bar — ada ikon **install** (⊕ atau monitor dengan panah)
2. Klik ikon tersebut
3. Klik **"Install"**

#### Cara C: Via Menu Browser
- **Chrome**: ⋮ → **"Install TAMIMA CL..."**
- **Edge**: ⋯ → **Apps** → **"Install this site as an app"**

### Langkah 3: ✅ Selesai!

Aplikasi otomatis terpasang dan muncul di:
- 📂 **Start Menu** → ketik "TAMIMA" untuk cari
- 🖥️ **Desktop** (shortcut)
- 📌 Bisa di-**pin to Taskbar**

## 🆘 Troubleshooting

### Tombol "Install App" tidak muncul
- Pastikan menggunakan **Chrome** atau **Edge** (bukan Firefox/Safari)
- Refresh halaman (F5)
- Pastikan diakses via HTTPS (bukan http)
- Cek koneksi internet pertama kali

### "Install App" sudah di-klik tapi tidak ada respon
- Cek apakah ada popup blocker
- Buka di Incognito/InPrivate mode dan coba lagi

### Aplikasi tidak bisa offline
- Buka sekali secara online untuk caching pertama
- Tunggu 5-10 detik setelah load (Service Worker registering)
- Restart browser, lalu coba offline lagi

### Data hilang
- Data tersimpan di LocalStorage browser
- **Jangan clear browser data** untuk situs ini
- Backup berkala via tombol **"Save Resource HTML"** atau **Export Excel**

## 💡 Tips Penggunaan

1. **Pin to Taskbar** untuk akses cepat:
   - Klik kanan ikon TAMIMA di Start Menu
   - "Pin to taskbar"

2. **Buat Shortcut Desktop**:
   - Otomatis dibuat saat install
   - Jika tidak, drag dari Start Menu ke Desktop

3. **Multi-Window**:
   - Aplikasi bisa dibuka di banyak window sekaligus
   - Tiap window punya state sendiri

4. **Print Profesional**:
   - Pakai Microsoft Edge untuk hasil print terbaik
   - Pilih kertas **Letter** (sudah default)
